## 毒鸡汤

毒鸡汤存放在 soul.json 和 soul.sql 文件中

注：数据来源于网络


## 接口示例

执行 api.php 可获取随机一条 毒鸡汤 如：

```json
{
    "id": 1361,
    "content": "减肥这件事情吧，减的快反弹的快，减的慢放弃的快。",
    "created_at": "2019-11-12 06:08:42",
    "updated_at": "2019-11-12 06:08:42"
}
```
